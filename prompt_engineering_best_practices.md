# Prompt Engineering 最佳实践

## 核心概念
Prompt Engineering 是为大模型设计高质量提示词以获得可控、高准确度输出的一门技术。核心目标包括：
- 让模型理解你的意图（Intent）
- 控制输出结构（Format Control）
- 提升推理能力（Reasoning）
- 降低错误（Safety + Reliability）

## 最佳实践
1. **显式说明任务**
2. **提供结构化格式要求**
3. **约束内容与风格**
4. **提供上下文**
5. **分步骤提示（Step-by-step）**
6. **确认（Refine）循环**

## 示例 Prompt（通用模板）
```
你是一名专业的___领域专家，请完成以下任务：
任务：____
要求：
1. 输出格式：____
2. 风格：____
3. 限制：不得____
请开始。
```
